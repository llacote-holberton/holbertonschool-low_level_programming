# Group of projects about learning C basics

# Overview
This repository holds several exercices and
  small-scope projects aiming to teach through practice
  all the basic notions of programming as well as
  the specificities of C syntax.

# Structure
Each directory contains a set of exercices
  tailored around specific notions
  or a full project of reasonable functional scope.

NOTE: README will be updated once all projects are done
