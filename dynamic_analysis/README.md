# Dynamic Analysis

This project aims at teaching how to use Gnu Debugging Tool
  to improve code review and self-drived improvements on 
  code writing quality.

# Exercises frame

All exercises will be based on the analysis of 
  design and execution of the following source file.
https://github.com/hbtn-edu/public_resources/blob/main/3776-dynamic_analysis/source.c

## Constraints
- Provided file must be renamed as "sample.c"
- Provided source code must NOT be modified.
- All runs must use the following input exactly (without quotes): "123456".

## Additional infos and public_resources
- GDB doc: https://www.sourceware.org/gdb/documentation/
- Program must be compiled with -g flag to allow better analysis
  (-d for integrate debbuging symbols).
